# tcc-random-forest-npc

## Versão da Godot 
Utilize a versão [Godot 4.7 - .dotnet](https://godotengine.org/download/windows/).
Tenha o [.dotnet](https://dotnet.microsoft.com/pt-br/download/dotnet) instalado em seu computador também

O Addon/Plugin para a Godot está na pasta npc-godot, apenas copie e jogue em qualquer projeto criado na Godot para ativar

## GDD do Prototipo

Um simulador para os Npcs ficarem decidindo suas ações, o jogo vai ser 2D e os npcs 8 direções

### O que precisa ser feito
- [ ] Colocar animação para o NPC
- [ ] Configurar os estados para os NPCs agirem conforme eles
- [ ] Colocar para spawnar uma certa quantidade de NPCs
- [ ] Colocar a camera dinamica (dar zoom, se mover pelo cenario) ou camera fisica
- [ ] Ao clicar mostrar o estado atual do npc
- [ ] Criar um mundo para eles ficarem
